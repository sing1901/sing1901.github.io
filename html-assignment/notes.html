HTML Tags that i have learnt

<p>
paragraph tag is used to add a paragraph to an html page
<img>
The image tag is used to embed images in a web page.
<ul>
The unordered list tag is used to create a list of items with bullet points.
<table>
The table tag is used to create a table of data with rows and columns.
<a>
The anchor tag is used to create hyperlinks to other web pages or resources.
